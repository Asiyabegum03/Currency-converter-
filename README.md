# Currency Converter

A simple Currency Converter web application that converts one currency into another using live exchange rates.

## Features

* Convert currencies instantly
* Easy-to-use interface
* Real-time exchange rate data
* Fast and accurate conversion

## Technologies Used

* HTML
* JavaScript
* Exchange Rate API

## Files

```text
currency-converter/
├── index.html
├── script.js
└── README.md
```

## How to Use

1. Open `index.html` in a web browser.
2. Enter the amount to convert.
3. Select the source currency.
4. Select the target currency.
5. Click the Convert button to see the result.
